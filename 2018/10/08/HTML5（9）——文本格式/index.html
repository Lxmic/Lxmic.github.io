HTML5（9）——文本格式

发表于 2018-10-08 更新于 2022-07-31 分类于 HTML5入门 阅读次数：
本文字数： 1.2k 阅读时长 ≈ 2 分钟

HTML格式元素

与之前学习的style 属性其实有相同的功能，而HTML自己也有特殊的文本格式特定的元素。下面是HTML中格式元素：

  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <mark> - Marked text
  • <small> - Small text
  • <del> - Deleted text
  • <ins> - Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text

b和strong元素

两者在网页中均显示为粗体，但是两者的区别在于前者只是样式的改变，而后者是具有语义上的强调。

斜体元素

<i> 和<em>在HTML中都是斜体，但是后者同样是由语义 上的强调

small元素

<small>是用来定义更小的文本。<h2>HTML <small>Small</small> Formatting</h2>

mark元素

它是用来标记文本的，例如：
<h2>HTML <mark>Marked</mark> Formatting</h2>

del元素

del是用来画横线删除文字，但文字已然存在，也算是一种修改方式。<p>My favorite color is <del>blue</del> red.</p>

ins元素

用下划线来表示文本的插入。
<p>My favorite <ins>color</ins> is red.</p>

sub元素

它是将文字显示成靠下的形式，我们所说的下标可以用<sub来实现>。
<p>This is X<sub>2</sub> text.</p>

sup元素

相对地，<sup>元素则是用来表示上标。
<p>This is X<sup>2</sup> text.</p>
