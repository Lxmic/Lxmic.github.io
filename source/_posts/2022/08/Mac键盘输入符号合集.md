# <font color=green>常用符号快捷键</font>
英文输入法状态下，我个人常用的几个符号
- 无穷∞： option + 5
- 微µ： option + m
- 摄氏度°C： shift + option + 8，再输入大写的C
- delta ∆： option + j 
- ±： shift + option + =
- ≥： option +  >
- ≥： option +  <

{% note info %} option条件下，有些按键出现两个符号，例如右上角`=/+`这个按键出现≠和±两符号，其中右下角≠：option + `+/=`，而左上角±，则shift + option + `+/=` ，这是一个规律，其他按键都适合。{% endnote %}
